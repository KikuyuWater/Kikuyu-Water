# Careers Data

This folder contains career opportunities and job postings for Kikuyu Water Company.

## Structure

Add job posting data and related documents here.

## Job Posting Format

Each job posting should include:
- Position title
- Department
- Job description
- Requirements/qualifications
- Application deadline
- How to apply
- Job type (Full-time, Part-time, Contract, Internship)
- Location

## Example

```javascript
{
  id: 1,
  title: "Water Quality Analyst",
  department: "Technical",
  description: "Responsible for monitoring and analyzing water quality...",
  requirements: [
    "Bachelor's degree in Chemistry or related field",
    "3+ years experience in water quality testing",
    "Knowledge of WASREB standards"
  ],
  deadline: "2026-02-15",
  type: "Full-time",
  location: "Kikuyu",
  applicationEmail: "careers@kikuyuwater.co.ke"
}
```

Place job-related documents (application forms, job descriptions PDFs) in this folder.
